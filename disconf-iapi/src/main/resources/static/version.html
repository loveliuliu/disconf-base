disconf iapi 发布日志
v1.0.0 2016-11-09 1.从disconf-web中分离出来，保持稳定 2.主要提供几个接口：    1.获取zoo host : /api/zoo/hosts    2.获取zoo prefix : /api/zoo/prefix (zookeeper上 /disconf 节点)    3.获取配置项 : /api/config/item    4.获取配置文件 : /api/config/file    5.获取所有配置或指定Key的Meta : /api/config/metas    6.获取所有配置项的值(不包含配置文件) : /api/config/item/values
